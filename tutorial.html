序章 · 棋盘与兵种

六角棋盘

每场战斗在六角格棋盘上进行。棋盘分为上下两半，蓝方（玩家）部署在下半区，红方（AI）部署在上半区，中间以河流/分界线隔开。

概念说明
🗺 棋盘 12×8 六角格棋盘。六角格使用立方体坐标 (q,r,s)，近战只能攻击相邻格，远程根据武器射程攻击。
🚩 部署 选兵后进入布阵阶段，拖拽己方部队到己方半场任意位置。确认后进入自动布阵（AI 同时部署）。
🎯 攻击范围 近战武器攻击范围为 1 格（相邻）；弓弩远程武器范围为 2-4 格，可越过中间单位射击。
🏃 移动 兵团每回合按移动力值移动对应格数。移动后若相邻敌军，可发动攻击。
🔄 回合 玩家先行动→AI 行动→回合结束。每个兵团每回合可选择移动+攻击 或 原地攻击。

兵团类型

游戏中共有五种基础兵团类型，每种有其独特的战场定位。

⚔️

步兵（Infantry）

基础近战单位。人数最多，可持盾，适合正面战线。可装备剑/斧/长柄等近战武器。克制关系：正面抵抗骑兵冲锋（长柄武器附带"抵御冲锋"）。

🐴

骑兵（Cavalry）

高机动近战单位。人数较少但移动力高，适合侧翼包抄与绕后。冲锋时对步兵造成额外士气打击。注意：骑射手是特殊变种——可装弓/弩远程输出，但只能通过召唤获得。

🏹

远程兵（Archer）

远程输出单位。可在安全距离外持续射击。弓类惧怕近身（相邻格有敌军时无法射击），弩类无视近战干扰。面对高护甲目标时可用重甲克制武器。

🦅

空军（Flying）

空中单位。可跨越地形障碍，无视地面阻挡。攻击地面步兵时触发"空中震慑"——额外扣除步兵士气。分为近战空军（剑/斧）和远程空军（弓/弩，需通过召唤获得"骑射"武器）。

🐻

野兽（Beast）

动物单位。只能装备护甲，无法使用武器、盾牌或坐骑。天生拥有强大的基础攻击力与血量，但士气较低，容易溃逃。野兽有额外体型规模限制。

第一章 · 基础

认识你的兵团

每个兵团由种族、武器、盾牌、护甲、坐骑五要素构成，最终合成为战场上的实际属性。

属性含义来源
🛡 总兵力 兵团的整体血量池 = 单兵HP × 士兵数。被打到 0 即溃逃。 种族基础HP + 坐骑加成；受机动惩罚影响
⚔ 杀伤 每次攻击的基础伤害值，扣除对方护甲后造成实际伤害。 武器基础伤害，或种族天生武器（空手近战）
🔨 破甲 最低保底伤害——即使护甲高于杀伤，也能打出破甲值。 武器破甲属性，空手为 0
🛡 护甲 从敌方杀伤中直接减免的伤害量（先减后算保底）。 种族天生护甲 + 护甲装备 + 坐骑护甲 + 盾牌附魔
🎯 攻击距离 射程——能打到几格外。1 为近战，2-4 为远程。 武器 allowedRange；弓惧怕近身，弩无视近战干扰
👥 攻击范围 单次攻击同时打击多少敌方士兵（与军团规模挂钩）。 max(军团规模-1, 武器attackRange)；大型单位打击面更大
🏃 移动 每回合可移动的六角格数量。重甲会惩罚移动力。 max(种族移动, 坐骑移动) + 护甲机动惩罚；骑兵/空军保底 2 格
💪 士气 战斗意志。降到 0 则溃逃。受背袭/侧袭/战损/连锁溃逃影响。 种族基础士气；未行动的回合结束 +3；脱离包围 +5
🏹 远程免伤 减免远程攻击的百分比。盾牌提供此效果。 盾牌效果（如"远程免伤+25%"）；双手武器无法持盾
📏 军团规模 兵团在棋盘上占据的空间 = 单位规模 × 人数。上限 160。 种族 scale + 坐骑 scale；超过 160 则自动压缩人数

第三章 · 召唤

召唤之门

召唤是获取独特部队的核心途径——AI 会根据你选择的世界设定和品阶，为你量身打造一支独一无二的兵团。

🌍

八大世界设定

战锤、DND、奇迹时代、魔兽、指环王、暗黑哥特、中世纪欧洲、中国古代——每个世界有独特的种族池和野兽池。

✨

AI 深度生成

AI 会生成：部队名称、背景故事、信仰、武器名字/伤害/破甲/特效、护甲名字/防御、盾牌、坐骑——全部独一无二。

📊

威力校准

系统自动计算召唤结果的威力指数（进攻分 + 生存分），若超出品阶范围则自动调整属性，确保平衡。

💬

自定义描述

你可以在召唤时输入自定义描述（如"我想要一支擅长冲锋的重骑兵"），AI 会据此生成更符合你期望的部队。

🔧 API 配置

游戏默认使用 DeepSeek API（deepseek-v4-flash）。你可以在设置中配置自己的 API Key，支持多配置文件切换。
API 端点：POST /api/summon —— 30 秒超时；POST /api/ai-battle-generate —— 45 秒超时。

第四章 · 对战

对战基本操作与流程

从选兵到胜利，一场完整的对战分为四个阶段。理解流程是制胜的第一步。

掷骰争先后

双方各掷 1d6。点数高者先手，平局随机决定。先手在部署和战斗中均优先行动。

交替部署

双方轮流将部队从预备栏放置到棋盘己方半场。点击预备栏部队 → 点击空六角格即可放置。双击已放置部队可回收。

战斗回合

选中己方部队 → 点击绿色高亮格移动 → 点击红色高亮敌方格攻击。每回合每支部队可移动一次并攻击一次。完成后点"结束回合"。

结算奖励

全歼敌方或敌方全溃 = 胜利。获得积分 + 敌方装备掉落（首通必掉保底装备）。积分可用于购买新部队或召唤。

⚠ 关键规则

· 弓兵被近战贴脸时无法射击（惧怕近身），弩兵无视此限制
· 移动时可能触发敌方偷袭（概率 10%-40%，取决于偷袭者品阶）
· 空军永远不会被偷袭，但士气归零仍会溃逃
· 敌方回合时你无法操作，请等待 AI 行动完毕

第五章 · 界面

板块与界面介绍

游戏主界面共有六大功能板块，从备战页一键触达。

🏕️

兵营

展开查看每支部队的详细属性与装备槽位。点击装备槽可更换/卸下装备。部队按品阶分类展示。

📦

背包

查看和管理你拥有的装备（武器/盾牌/护甲/坐骑）。可在此出售不需要的装备换取积分。

🗺️

开始对战（关卡模式）

四个难度 × 六个关卡 = 24 个预设战役。首通有保底装备掉落。选择难度和关卡后进入布阵。

🤖

AI 对战

与 AI 生成的对手交战。消耗部队但可获得积分和装备掉落奖励。无限重复挑战。

🛒

商城

购买新部队或装备。分为"兵种"和"装备"两个标签页。已购买的装备进入背包，需手动装备到部队上。召唤过的兵团和装备也会在商城中解锁（价格更贵）。

🌌

召唤之门

AI 生成独特部队。你可以自定义世界观设定（战锤、DND、魔兽、指环王等八个世界），选择品阶和输入描述，让 AI 为你量身打造独一无二的兵团。召唤的种族和装备均独立生成并可在商城再次购买。

第六章 · API

API 接口使用

游戏运行在本地服务器（默认端口 3456），所有数据通过 REST API 管理。

方法路径用途
POST/api/summonAI 生成部队 —— 发送品阶、种族、世界设定、自定义描述，返回完整部队定义
POST/api/ai-battle-generateAI 生成对手 —— 返回敌方部队列表、装备、种族等完整配置
GET/api/saves获取所有存档列表
PUT/api/saves/:name保存/更新存档（JSON body）
GET/api/summoned/:name获取某存档的召唤数据（单独存储以减少主存档体积）
GET/PUT/api/api-config管理 AI API 密钥和多配置文件
GET/PUT/api/settings用户设置（壁纸偏好等）
GET/POST/DELETE/api/wallpapers管理自定义壁纸
🔑 配置 API Key

在游戏主菜单的设置页面输入你的 DeepSeek API Key，或者替换为其他兼容 OpenAI 格式的 API 端点。
配置文件存储在 assets/api_config.json，支持多个 API 配置文件之间切换。

进阶 · 战术

走位、士气与战斗艺术

掌握了基础之后，这些进阶技巧将让你在战场上所向披靡。

🎯 走位——攻击角度的力量

  • 正面攻击：无加成，标准伤害
  • 侧袭（侧面）：护甲 -10%，士气 -5，骑兵伤害 ×1.2
  • 背袭（后方）：护甲 -20%，士气 -10，骑兵伤害 ×1.4
  • 棋子面朝方向由最后一次移动或攻击决定
  • 骑兵冲锋的动能加成取决于实际移动距离（2-4 格 → 1.5-2.0×）

💔 打击士气——不战而屈人之兵

  • 包围：3+ 敌方邻接 → 每回合 -5 士气
  • 侧袭 -5 士气，背袭 -10 士气
  • 友军溃逃：2 格内每支溃逃友军 -3 士气（连锁效应）
  • 血量 < 75%/50%/25% → 士气 -10/-15/-25
  • 空军攻击步兵额外 -5 士气（空中震慑）
  • 击杀敌方：5 格内友军获得 +5 士气

⚙️ 换装备操作

  • 在兵营展开部队详情 → 点击装备槽 → 选择背包装备即可换装
  • 装备先在商城购买进入背包，再到兵营装备到部队
  • 双手武器会自动卸下盾牌；野兽单位只能穿护甲
  • 装备兼容性由兵种类型和体型决定（如"仅步兵"、"中型以上"）
  • 迅捷特性：武器效果，血量 -20% 但移动力 +1（无视上限）

🛡 战斗注意事项

  • 高护甲 vs 高破甲：破甲值是保底伤害，护甲再高也能打出破甲值
  • 弩兵克制重甲（护甲 ≥ 30 时伤害 ×1.2）且无视近战干扰
  • 长柄武器附带"抵御冲锋"——正面接骑兵时反骑额外伤害 ×1.5
  • 军团规模上限 160：超限部队会被压缩人数，同等兵力下更抗打
  • 布阵时近战在前、远程在后，AI 在困难和传说难度也会这样布阵

⚡ 偷袭机制

  • 任何部队移动时，周围 1 格的敌方近战单位可能发动偷袭
  • 触发概率：黑铁 10% / 青铜 20% / 黄金 30% / 钻石 40%
  • 偷袭伤害 = 兵团 50% 人数 × 武器伤害 × 方位倍率（侧袭 1.15×，背袭 1.30×）× 60%
  • 偷袭附带 -5 士气打击，可能直接导致溃逃
  • 空军永远不会被偷袭；骑兵冲锋后也不会

🌌 召唤兵团的经济系统

  • 召唤的兵团由召唤种族基底（属性/体型/天生护甲）与召唤装备（武器/护甲/盾牌/坐骑）两部分构成
  • 召唤过的兵团类型会在商城自动解锁，可以再次购买（价格比商城预设兵种更贵）
  • 召唤过的装备同样会在商城装备标签页解锁，供其他部队购买使用
  • 每个部队的装备槽位对应：武器（单手/双手/远程）、盾牌（仅单手武器可用）、护甲、坐骑
  • 卸下装备会放入背包，再从背包可装备到任意兼容的部队上

🔮 其他隐藏机制

  • 部队即使士气不归零，血量归零也会溃逃
  • 未行动的部队在回合结束时 +3 士气
  • 非战斗减员：军团规模越大，每点移动力额外消耗 3% 血量
  • 迅捷武器：+1 移动力但 -20% 血量
  • 独特装备（_unique:true）无法在商城购买，只能通过特定关卡首通获得
  • AI 在困难难度会集火最弱的玩家部队，优先切后排远程
难度参考

四个难度等级

AI 的行为和敌军配置在四个难度下截然不同。

🟢

简单

敌方多为农民兵。AI 直线冲向最近目标，无特殊走位，部署随机。
适合熟悉基础操作。

🔵

中等

敌方混编农民与正规兵。AI 主动进攻，弓兵拉扯走位。
需要一定的兵种克制意识。

🟠

困难

敌方全正规兵种，AI 精准集火最低血量单位。远程部署在最后排，骑兵绕后。
需要精通走位与士气打击。

🔴

极限

敌方精锐大军压境（10 支部队 vs 你的 6 支）。AI 穷追不舍不复退。通关 legend_6（万界霸主）可获得独特钻石武器「万界终结者」——基础杀伤 60、破甲 35、射程 4、附带迅捷。
